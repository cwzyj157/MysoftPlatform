示例代码如下： 

CbProductCbFtRule rule = new CbProductCbFtRule();

// 注意：调用Delete方法前，要给主键字段赋值。
rule.ProductCbFtRuleGUID = productCbFtRuleGUID;

// Delete方法能将对象从相应的数据表中删除。
rule.Delete();

执行Delete调用后，数据访问层将会生成这样一个参数化SQL，最终执行它： 

DELETE FROM cb_ProductCbFtRule
where ProductCbFtRuleGUID = @ProductCbFtRuleGUID


注意：
1. 在执行Delete之前，必须为主键对应属性赋值，否则不能生成Where条件。
2. 如果是复合主键，需要为多个主键对应属性赋值。
3. 由于只能根据主键执行删除操作，因此为非主键属性赋值无效。
