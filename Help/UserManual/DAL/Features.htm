.net 版本支持

  1. 新的数据访问API只支持 .net 3.5以上版本。
  2. VS2003的项目，如果需要使用，请升级.net framework.

调用存储过程

  1. 支持带返回值的命令参数。
  2. 支持从存储过程中获取一个实体列表。
  3. 支持从存储过程中获取一个实体对象。
  4. 支持从存储过程中获取一个标量值。
  5. 支持从存储过程中获取一个标量列表。
  6. 支持从存储过程中获取一个DataTable
  7. 默认情况下会自动生成SqlParameter数组，但也支持手工指定，满足未知需求。

执行参数化SQL语句

  1. 【不】支持带返回值的命令参数。
  2. 支持从参数化SQL得到一个实体列表。
  3. 支持从参数化SQL得到一个实体对象。
  4. 支持从参数化SQL得到一个标量值。
  5. 支持从参数化SQL得到一个标量列表。
  6. 支持从参数化SQL得到一个DataTable
  7. 默认情况下会自动生成SqlParameter数组，但也支持手工指定，满足未知需求。

连接与事务

  1. 用ConnectionScope创建一个作用域，实现多个操作共享连接。
  2. 用ConnectionScope创建一个作用域，实现事务支持。
  3. ConnectionScope允许嵌套。
  4. 在事务的作用域范围内，事件的订阅者也会进入事务控制范围内。

实体要求与建议

  1. 建议将所有数据实体定义到专门的类库项目中，文件名以 .Entity.dll 结尾
  2. 建议所有数据实体从BaseEntity继承过来。
  3. 实体的数据成员只支持属性，不支持字段。
  4. 实体不支持嵌套。
