<static> {object} MapExt.parseJSON(strJson)
将JSON字符串反序列化为JSON对象.

1.如果JSON字符串格式存在错误,本函数将产生异常。
2.对于服务端返回的HttpResult类的JSON字符串.请使用parseJSONResult方法。
3.对于服务端返回的HttpResult类的XML字符串,请使用parseXMLResult方法。
4.以上两个方法不会产生异常,解析失败也会返回HttpResult对象,只是对象的Result属性为false。 
示例代码:
var str = "{ ContractName : "合同1", Amount: 99.76 }";
var json = MapExt.parseJSON(str);
if (json){
  alert(json.ContractName);
}
参数列表:
{string} strJson
- 表示被转换的JSON字符串
返回值:
{object} 转换后的js对象
<static> {HttpResult} MapExt.parseJSONResult(strJSON)
将JSON字符串转换为HttpResult对象.

1.请不要使用本方法来反序列化其他格式的JSON字符串。
2.本方法不抛出异常,解析失败也会返回{Result : false, ErrorMessage : "具体的错误原因"}。 
示例代码:
var str = "{ Result : true, KeyValue: "0D85EC64-18FB-4F61-ACD1-E0EBAD69A557" }";
var json = MapExt.parseJSONResult(str);
if (json.Result){
    alert(json.KeyValue);
}
参数列表:
{string} strJSON
- 表示被转换的JSON字符串
返回值:
{HttpResult} 对象实例
<static> {string} MapExt.toXml(rootName, jsObject)

var data = {name : "admin", vaule : "管理员"};
var xml = MapExt.toXml("user", data);

参数列表:

- 根节点名称
{object} jsObject
- 表示要被序列化的js对象
返回值:
{string} 过滤后的字符串
